Politique de Confidentialité de Chronos

Dernière mise à jour : [30/11/2024]
La protection de votre vie privée est importante pour nous. Cette politique de confidentialité décrit quelles informations sont collectées par Chronos, comment elles sont utilisées et vos droits concernant ces données.

1. Informations collectées
Chronos peut collecter les types de données suivants pour fonctionner correctement :

Identifiants utilisateur : ID Discord des utilisateurs pour exécuter certaines commandes.

Messages ou commandes : Chronos stocke temporairement les commandes envoyées pour les traiter.

Données de serveur : Nom du serveur, identifiants des canaux, paramètres spécifiques (si applicable).

Statistiques d'utilisation : Données anonymes sur la fréquence d'utilisation des fonctionnalités.Aucune information sensible ou personnelle n'est collectée.

2.Utilisation des informations Les données collectées par Chronos sont utilisées pour :Fournir les fonctionnalités du bot.Améliorer les services et corriger d'éventuels bugs.Générer des statistiques d'utilisation anonymes pour optimiser l'expérience utilisateur.

3. Partage des données Nous ne partageons vos données avec aucune tierce partie, sauf dans les cas suivants :
Obligation légale : Si la loi nous y oblige.
Problèmes techniques : Si un prestataire technique nécessite des informations pour résoudre un problème.

4. Durée de conservation Les données de session (commandes, messages temporaires) sont supprimées après leur traitement.Les identifiants utilisateur et autres informations nécessaires au fonctionnement des services peuvent être conservés jusqu’à ce qu’ils ne soient plus nécessaires ou sur demande de suppression.

5. Sécurité des données Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles pour protéger vos données contre tout accès non autorisé, modification, divulgation ou destruction.

6. Vos droits En tant qu'utilisateur, vous avez les droits suivants :
Accès à vos données : Vous pouvez demander à voir quelles données sont collectées.
Demande de suppression : Vous pouvez demander la suppression de vos données à tout moment.
Révocation du consentement : Retirez le bot de votre serveur pour arrêter la collecte de données.Pour exercer ces droits, veuillez contacter l'administrateur via le serveur support.7. Modifications de cette politiqueNous pouvons mettre à jour cette politique de confidentialité de temps en temps. Toute modification sera communiquée via un message sur le serveur support ou d'autres moyens appropriés.

Merci de faire confiance à Chronos !
